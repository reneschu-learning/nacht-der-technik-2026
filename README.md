# Nacht der Technik 2026 @Microsoft - GitHub Copilot
Willkommen zur Nacht der Technik 2026 bei der Microsoft Deutschland GmbH in Köln! 🎉

In dieser spannenden Session unter dem Motto "Democratizing Software Development" (zu Deutsch: "Softwareentwicklung für alle zugänglich machen") wirst du die Gelegenheit haben, mit GitHub Copilot ✈️ zu experimentieren und deine eigenen kleinen Web-Applikationen zu erstellen. Du hast noch nie programmiert? Kein Problem! Copilot wird dir dabei helfen, Schritt für Schritt deine Ideen 💡 in Code umzusetzen. 🚀 Nach einer kurzen Einführung in GitHub Copilot legst Du auch schon selbst los.

### Nicht bei der Nacht der Technik (gewesen)?
Wenn Du nicht bei der Nacht der Technik 2026 warst, GitHub Copilot aber trotzdem mal ausprobieren möchtest, brauchst Du folgendes auf Deinem Rechner:

- **Visual Studio Code** (z.B. installierbar über WinGet mit dem Befehl `winget install Microsoft.VisualStudioCode`)
  - Um die Emojis in diesem Text auch in Visual Studio Code zu sehen, brauchst Du noch eine Erweiterung, z.B. die **:emojisense: von Matt Bierner** (z.B. installierbar über die Kommandzeile `code --install-extension bierner.emojisense`)
- **Node.js** und **npm**, um das Programm lokal laufen zu lassen (z.B. installierbar über WinGet mit dem Befehl `winget install OpenJS.NodeJS`)
- Optional aber empfohlen: **TypeScript** (installierbar **nach** der Installation von Node.js und npm über den Befehl `npm install -g typescript`)
- Einen GitHub Account - wenn Du keinen hast, kannst Du kostenlos einen unter https://github.com erstellen. Klicke einfach oben rechts auf `Sign up`. Und um möglichst sicher zu sein, richte am besten gleich einen PassKey ein 🔑
- Eine GitHub Copilot Lizenz. Schau einfach mal unter https://github.com/features/copilot/plans vorbei und wähle eine passende Optionen. Es gibt sowohl eine eingeschränkte kostenlose Version als auch kostelose Versionen für Schüler und Studenten (siehe https://github.com/education/students). Oder wählst eine der kostenpflichtigen 💸 Optionen, mit denen Du dann nachher richtig weiterarbeiten kannt.

Du möchtest lieber nichts auf Deinem Rechner installieren? Dann probiere GitHub Codespaces aus! Klicke einfach oben rechts auf den Pfeil in dem grünen Button `Code`, wechsle auf das `Codespaces` Tab und klicke auf `Create codespace on main`. **Hinweis:** Die Verfügbarkeit der Option hängt von Deinem GitHub Account ab.

### Bereit? - Dann folge diesen Schritten

1. Öffne **Visual Studio Code (VS Code)** auf deinem Laptop. 💻 - Wenn Du vor Ort im Raum sitzt, sollte das schon jemand für Dich gemacht haben 😉
2. Öffne das **GitHub Copilot Chat Panel**, indem Du oben mittig auf das GitHub Copilot Symbol klickst. Falls das Panel schon offen ist, ist alles okay. ✅  
   ![GitHub Copilot Chat Panel öffnen](.assets/CopilotChatIcon.png)  

3. Sollte im Chat Panel noch ein alter Chat sichtbar sein, klicke auf das Plus-Symbol (+) oben rechts, um einen **neuen Chat** zu starten. 🆕  
   ![Neuen Chat starten](.assets/NewChatSession.png)  

4. Klicke in das Chatfenster ganz unten und tippe `/ndt` (für Nacht der Technik) ein und drücke `Enter`.  
   ![Session starten](.assets/StartSession.png)  

5. Jetzt geht's los! 🚀 Befolge die Anweisungen von Copilot, stelle ihm Fragen, gib ihm Ideen. Es ist **Deine** Applikation - mach' was draus! 💪  
   ![Los geht's!](.assets/RunningSession.png)

Du bist vor Ort im Raum und hast Fragen, Wünsche oder hängst fest? Wir sind für Dich da! Sprich uns einfach an. 😊 Oder noch besser: zeig uns, was Du cooles gebaut hast! 🤩
